## Requirement
1. Install nodeJS [here](https://nodejs.org/en/download/)
2. Make sure no process running on port **8888**

## Install
```
npm install
```

## Start the simulator
### With log (to log/ folder)
```
npm start
```

### Without log
```
node server.js
```
